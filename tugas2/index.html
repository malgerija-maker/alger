01. Pengertian OOP

Object Oriented Programming (OOP) Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek, dimana program dibuat menggunakan class dan objek. Dalam OOP terdapat beberapa konsep utama seperti Encapsulation, Inheritance, Polymorphism, dan Abstraction yang membantu membuat program lebih terstruktur, modular, dan mudah dikembangkan.

02. Konsep Dasar

  • Class: Blueprint atau cetakan untuk membuat objek.
  • Object: Instance nyata yang tercipta dari sebuah class.
  • Attribute: Representasi data melalui variabel di dalam class.
  • Method: Logika fungsional atau fungsi di dalam class.

03. Class Sederhana (Kereta)

Pada contoh program di bawah ini terdapat class Kereta yang memiliki atribut nama dan rute untuk menyimpan informasi perjalanan publik. Objek ini memiliki method berangkat() dan tiba() untuk mensimulasikan pergerakan transportasi rel.

DART SOURCE CODE
// Nama   : M. Algerija hertyaz paslam
// Nim    : 251410021
// Kelas  : SI2A

class Kereta {
  String? nama;
  String? rute;
  
  void berangkat() {
    print('Kereta $nama tujuan $rute sedang bersiap berangkat.');
  }
  
  void tiba() {
    print('Kereta $nama telah sampai di stasiun $rute.');
  }
}

void main() {
  Kereta keretaAlger = Kereta();
  
  keretaAlger.nama = "Kertajaya";
  keretaAlger.rute = "Jakarta";
  keretaAlger.berangkat();
  keretaAlger.tiba();
}

07. Encapsulation

Enkapsulasi diterapkan pada class AkunBank untuk melindungi variabel _saldo agar tidak bisa diubah langsung dari luar, melainkan harus melalui method setorTunai().

DART SOURCE CODE
// Nama   : M. Algerija hertyaz paslam
// Nim    : 251410021
// Kelas  : SI2A

class AkunBank {
  String pemilik;
  double _saldo; 

  AkunBank(this.pemilik, this._saldo);

  void setorTunai(double jumlah) {
    if (jumlah > 0) _saldo += jumlah;
  }
  
  double get totalSaldo => _saldo;
}

void main() {
  AkunBank userAlger = AkunBank("Alger Paslam", 500000);
  print('Pemilik Akun: ${userAlger.pemilik}'); 
  userAlger.setorTunai(250000);
  print('Total Tabungan: Rp ${userAlger.totalSaldo}'); 
}

09. Product Class (Kursus)

Class KursusOnline mengandalkan "Named Parameters" untuk mendefinisikan detail materi. Sistem ini membuat kode lebih fleksibel saat menambahkan info mentor, durasi, atau harga materi.

DART SOURCE CODE
// Nama   : M. Algerija hertyaz paslam
// Nim    : 251410021
// Kelas  : SI2A

class KursusOnline {
  final String judul, mentor; 
  final int durasiMenit; 

  KursusOnline({
    required this.judul,
    required this.mentor,
    required this.durasiMenit,
  });
}

void main() {
  KursusOnline masterclass = KursusOnline(
    judul: "Flutter Advanced",
    mentor: "Alger Paslam",
    durasiMenit: 1200,
  );

  KursusOnline bootcamp = KursusOnline(
    judul: "Dart System Design",
    mentor: "M. Algerija",
    durasiMenit: 800,
  );

  print('Materi 1: ${masterclass.judul} by ${masterclass.mentor}'); 
  print('Materi 2: ${bootcamp.judul}'); 
}
